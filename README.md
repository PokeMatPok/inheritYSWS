# Inherit YSWS

**Give abandoned open source projects a second life.**

[Visit Inherit](https://inherit.dino.icu/) • [Visit the backend repo](https://github.com/PokeMatPok/inheritYSWS-backend/)

## What is Inherit?

Inherit is a Hack Club YSWS program that revives abandoned open source projects. Hack Clubbers adopt forgotten repos, ship new features, get peer-reviewed, and earn rewards (Github Badger, stickers, books) or donate to charity.

### How it works:
1. **Browse** abandoned Hack Club projects
2. **Claim** a project and set your prize goal
3. **Build** features and track time with Hackatime
4. **Submit** for peer review
5. **Receive** your reward when merged

## Tech Stack

- **Frontend**: SvelteKit + Vite
- **Backend**: Express.js
- **Hosting**: Vercel
- **DNS**: hackclub/dns

## Development
```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
